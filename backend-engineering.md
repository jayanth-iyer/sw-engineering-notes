# What to Study in Backend Engineering?

1. Event-Driven Architecture - Services talk via events.
2. Observability - Logging, Monitoring, Tracing.
3. Circuit Breakers - Prevent cascading failures.
4. Sharding and Partitioning - Distributing data across nodes.
5. Rate Limiting - Control traffic to services. Token Bucket.
5. Authentication.
    - Session-based Auth.
    - Token-based Auth (JWT).
    - OAuth2.0 Flow.
    - API Key Auth.
    - Multi-factor Auth.
    - SSO (Single Sign-On).
